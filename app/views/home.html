Bienvenido a CIDA: Continuous Integration Demo App

Ésta es una aplicación web que he creado para poner en práctica tecnologías relacionadas con la Integración Continua, además de otras tantas relacionadas con el Desarrollo Ágil de aplicaciones Web en AngularJS.

En el lado de las tecnologías para la Integración Continua, las que han motivado el desarrollo de esta aplicación web, tenemos:

  • Brunch
  • NPM
  • Bower
  • Sass
  • AngularJS
  • ng-docs

Aprovechando la ocasión he introducido una serie de tecnologías con las que deseaba experimentar en esta tipo de Arquitecturas. Dichas tecnologías son:

  • NodeJS
  • SequelizeJS
  • PostgreSQL

A todas estas tecnologías hay que sumarle el hecho de que el despliegue lo voy a realizar en Heroku.

En una instalación de un Apache en un servidor propio sólo tendríamos que compilar el proyecto y realizar el despliegue sobre el servidor, para lo que se suelen utilizar herramientas como Jenkins o Travis.

Sin embargo, para hacer uso de Heroku habrá que realizar una serie de configuraciones específicas para admitir esta propuesta de solución para la Integración Continua para aplicaciones Web con AngularJS.
