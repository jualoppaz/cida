Bienvenido a CIDA: Continuous Integration Demo App

Ésta es una aplicación web que he creado para poner en práctica tecnologías relacionadas con la Integración Continua, además de otras tantas relacionadas con el Desarrollo Ágil de aplicaciones Web en lenguaje JavaScript.

En el lado de las tecnologías para la Integración Continua, las que han motivado el desarrollo de esta aplicación web, tenemos:

  • Brunch
  • NPM
  • Bower
  • Sass
  • AngularJS
  • ng-docs
