Sass

Esta aplicación consta de una única hoja de estilos, la cual es generada de forma dinámica mediante Sass en tiempo de compilación a partir de muchas otras hojas de estilos específicas.

Un aspecto fundamental de una aplicación web es el diseño. Sin embargo, resulta muy complicado mantener y modificar la maquetación de una aplicación web sólo con CSS.

En una aplicación web tradicional podríamos tener una hoja de estilos inmensa, o por el contrario muchas hojas de estilo:

  • En el primer caso sería muy costoso realizar una buena maquetación e imposible la reutilización de código.
  • En el segundo caso, por cada nuevo módulo tendríamos que añadir una nueva importación a nuestro archivo HTML de nuestra aplicación web: 
    ...
    <link rel="stylesheet" type="text/css" href="/css/modulo1.css">
    <link rel="stylesheet" type="text/css" href="/css/modulo2.css">
    ...
    <link rel="stylesheet" type="text/css" href="/css/moduloN.css">
    ...

Para eso están los preprocesadores como Sass. Gracias a esta herramienta podemos definir variables, definir los estilos de una pantalla en forma de árbol anidando los selectores, añadir las media queries mediante funciones... En definitiva, podremos realizar una buena maquetación con una cantidad de líneas de código bastante inferior y centralizando muchísimos valores, evitando así código duplicado.

De este modo, no tendríamos que añadir una importación de una hoja de estilos por cada nuevo módulo que desarrollemos. Un ejemplo podría ser el siguiente: 

    ...
    <link rel="stylesheet" type="text/css" href="/css/vendor.css">
    <link rel="stylesheet" type="text/css" href="/css/app.css">
    ...

Para este proyecto la estructura de directorios utilizada para las hojas de estilos ha sido la siguiente: 

    styles/
        components/
            _breadcrumbs.scss
            _toast.scss
        layouts/
            _footer.scss
            _header.scss
            _main.scss
            _vnav.scss
        lib/
            _animaciones.scss
            _global.scss
            _globalLasts.scss
            _mixins.scss
            _variables.scss
        views/
            _sass.scss
        styles.scss

El fichero principal es el styles.scss, el cual incluye todos los ficheros parciales, y su contenido es el siguiente: 

    // Librerías
    @import "lib/animaciones";
    @import "lib/variables";
    @import "lib/mixins";
    @import "lib/global";

    // Componentes
    @import "components/breadcrumbs";
    @import "components/toast";

    // Layouts
    @import "layouts/header";
    @import "layouts/vnav";
    @import "layouts/main";
    @import "layouts/footer";

    // Vistas
    @import "views/sass";

    // CSS necesario al final
    @import "lib/globalLasts";
