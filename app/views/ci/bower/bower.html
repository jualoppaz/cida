Bower

Al igual que utilizamos NPM para la definición de dependencias de las herramientas utilizadas en este proyecto, utilizaremos Bower para la definición de dependencias para el Front-End.

La definición de dependencias explícitas para el Front-End, así como Bootstrap o Fontawesome, se realiza mediante el fichero bower.json. El contenido fundamental de este archivo es el siguiente: 

    ...,
    "dependencies": {
        "jquery": "^3.1.1",
        "angular": "^1.6.3",
        "bootstrap": "^3.3.7",
        "angular-bootstrap": "^2.5.0",
        "angular-ui-router": "^0.4.2",
        "angular-toastr": "^2.1.1",
        "angular-utils-ui-breadcrumbs": "^0.2.2",
        "components-font-awesome": "^4.7.0",
        "angular-sanitize": "^1.6.3",
        "angular-animate": "^1.6.3",
        "angular-touch": "^1.6.3"
    },
    "resolutions": {
        "angular": "^1.6.3"
    },
    ...

Estas dependencias son descargadas e instaladas en cada despliegue en Heroku tal y como se detalla en la sección de NPM.
