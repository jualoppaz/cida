Brunch

El Front-End de esta aplicación web se compila al completo con el framework Brunch.

Ésta es la herramienta clave para conseguir implementar una buena Integración Continua sobre una aplicación web con AngularJS. Podríamos haber elegido otra como pueden ser Gulp o Grunt, pero Brunch nos permite realizar multitud de tareas sin tener que desarrollar lo más mínimo gracias a la multitud de plugins existentes y sus amplias posibilidade de configuración.

La configuración de Brunch debe ir en un fichero brunch-config.js situado en la raíz de nuestro proyecto, ruta en la que iniciaremos nuestro servidor mediante Brunch.
